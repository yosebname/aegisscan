COMMAND INTERFACE:
================================================================================

aegisscan [OPTIONS] COMMAND [ARGS]

Commands:
  scan       - Perform network security scan
             - Options: targets, ports, type, timeout, concurrency, retries,
                        rate-limit, enrich, output, format, authorization flag
  
  import     - Import nmap XML results
             - Options: file, merge-with
  
  enrich     - Enrich scan results with additional data
             - Options: scan-run, banners, tls
  
  compare    - Compare scan results
             - Options: connect-vs-syn OR internal-vs-external OR runs
  
  report     - Generate report from scan results
             - Options: scan-run, format (html/pdf), output
  
  serve      - Start web server for AegisScan
             - Options: host, port, reload
  
  external   - Query external intelligence sources
             - Options: scan-run, provider (shodan/censys/all)

Global Options:
  --version        - Show version and exit
  --log-level      - Set logging level (DEBUG/INFO/WARNING/ERROR)
  --help           - Show help

USAGE EXAMPLES:
================================================================================

Basic scan:
  aegisscan scan --targets 192.168.1.0/24 --i-own-or-am-authorized

Scan with custom ports:
  aegisscan scan --targets 10.0.0.0/8 --ports 80,443,8080 \
    --i-own-or-am-authorized

Scan from file:
  aegisscan scan --targets targets.txt --ports 1-1024 \
    --i-own-or-am-authorized

Generate HTML report:
  aegisscan report --scan-run scan_abc123def456 --format html

Compare two scans:
  aegisscan compare --runs scan_123 scan_456

Query external data:
  aegisscan external --scan-run scan_abc123def456 --provider shodan
