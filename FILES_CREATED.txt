IMPORTER MODULE DETAILS
=======================

File: aegisscan/importer/nmap_importer.py

Classes:
- NmapScript(dataclass)
  * id: str - Script identifier
  * output: str - Script execution output

- NmapPort(dataclass)
  * port_id: int
  * protocol: str (tcp/udp)
  * state: str (open/closed/filtered)
  * service_name: Optional[str]
  * product: Optional[str]
  * version: Optional[str]
  * extra_info: Optional[str]
  * scripts: List[NmapScript]
  * to_dict() - Serialization method

- NmapHost(dataclass)
  * ip: str
  * hostname: Optional[str]
  * status: str (up/down)
  * ports: List[NmapPort]
  * os_matches: List[Dict] - OS detection candidates
  * to_dict() - Serialization method

- NmapScanResult(dataclass)
  * scanner: str = "nmap"
  * args: str - Command line arguments
  * start_time: Optional[datetime]
  * hosts: List[NmapHost]
  * stats: Dict - Scan statistics
  * to_dict() - Serialization method

- NmapImporter(class)
  * parse_file(filepath: str) -> NmapScanResult
    - Parses Nmap XML file
    - Validates file exists
    - Handles XML parsing errors
    - Returns complete scan result
  
  * parse_string(xml_string: str) -> NmapScanResult
    - Parses XML from string
    - Extracts metadata, hosts, stats
    - Handles multiple Nmap versions
    - Graceful degradation on missing fields
  
  * normalize_to_db(nmap_result, db_manager, scan_run_id)
    - Converts Nmap format to internal DB schema
    - Maps ports -> services
    - Includes OS candidates and metadata
    - Returns normalized dict for DB insertion
  
  * merge_with_scan(nmap_data, internal_scan_data, priority_rules)
    - Merges new Nmap data with existing scans
    - Priority rules: "newest", "nmap", "internal", "merge"
    - Intelligent service merging
    - Deduplication by IP and port
    - Returns merged dataset

Key Features:
- XML parsing with xml.etree.ElementTree
- Handles missing/partial data gracefully
- Version-agnostic (supports different Nmap versions)
- Comprehensive error logging
- Dataclass serialization support
- Smart merge logic with priority rules
- Full type hints throughout

USAGE EXAMPLES
==============

Nmap Import:
```python
from aegisscan.importer import NmapImporter

importer = NmapImporter()
result = importer.parse_file("/path/to/scan.xml")

# Normalize to DB format
normalized = importer.normalize_to_db(result, db_manager, "scan_123")

# Merge with existing data
merged = importer.merge_with_scan(
    result,
    existing_data,
    priority_rules={"services": "merge", "os_info": "newest"}
)
```

Shodan Lookup:
```python
from aegisscan.external import ShodanConnector

async with ShodanConnector(api_key="xxx") as connector:
    result = await connector.lookup_host("8.8.8.8")
    if result:
        print(f"Ports: {result.ports}")
        print(f"Vulns: {result.vulns}")
```

Censys Lookup:
```python
from aegisscan.external import CensysConnector

async with CensysConnector(api_id="xxx", api_secret="yyy") as connector:
    results = await connector.lookup_multiple(["8.8.8.8", "1.1.1.1"])
    for result in results:
        print(f"{result.ip}: {len(result.services)} services")
```
