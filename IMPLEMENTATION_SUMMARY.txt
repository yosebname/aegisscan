DATABASE ENTITIES (9 Total)
────────────────────────────────────────────────────────────────────────────

1. ScanRun
   - Represents scan execution with targets, config, version, status
   - Relationships: 1:N NmapObservation, 1:N DiffFinding
   - Indexes: 2 (composite + single)
   - Key fields: start_time, end_time, config_hash, status

2. Host
   - Core entity for discovered network hosts
   - Relationships: 6 outbound (Port, Service, Banner, TLSCert, etc)
   - Indexes: 3 (unique IP + composites)
   - Key fields: ip (unique), hostname, tags (JSON), temporal tracking

3. Port
   - Open/filtered ports on hosts with state info
   - Relationships: Belongs to Host
   - Indexes: 4 (composite unique + regular)
   - Key fields: port_number, protocol, state_connect, state_syn, rtt_ms

4. Service
   - Detected services with version info and confidence
   - Relationships: Belongs to Host
   - Indexes: 2 (composite)
   - Key fields: detected_service, product, version, confidence, source

5. Banner
   - Raw service banners/greetings
   - Relationships: Belongs to Host
   - Indexes: 2 (composite + temporal)
   - Key fields: raw_banner, parsed_fields (JSON), collected_at

6. TLSCert
   - X.509 certificate data from HTTPS services
   - Relationships: Belongs to Host
   - Indexes: 4 (composite + fingerprint unique + expiration tracking)
   - Key fields: subject, issuer, not_before, not_after, san_list (JSON)

7. NmapObservation
   - Nmap-specific findings and NSE scripts
   - Relationships: Belongs to Host, Belongs to ScanRun
   - Indexes: 2 (composite + scan_run)
   - Key fields: nmap_service, nmap_version, scripts_summary (JSON)

8. ExternalObservation
   - Multi-source intelligence (Shodan, Censys, etc)
   - Relationships: Belongs to Host
   - Indexes: 2 (composite + temporal)
   - Key fields: source, service, banner, raw_data (JSON)

9. DiffFinding
   - Changes/differences between scans
   - Relationships: Belongs to ScanRun
   - Indexes: 3 (scan_run, severity, finding_type)
   - Key fields: finding_type, severity, summary, evidence (JSON)
